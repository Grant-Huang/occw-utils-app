Aperçu du Système
Description du Système

Le Système de Devis OCCW est un outil professionnel de traitement de devis PDF conçu pour Oppein Cabinet Canada West Ltd., utilisé pour convertir les devis générés par le logiciel 2020 au format OCCW standardisé.

Description des Permissions

Permissions Utilisateur Régulier : Peut télécharger et analyser des PDF, voir les résultats, exporter des fichiers et consulter la documentation d'aide

Permissions Administrateur : En plus des permissions utilisateur régulier, peut également gérer les tables de prix OCCW, configurer les règles SKU et gérer les relations de mappage SKU

Fonctionnalités Principales
  • Support pour le téléchargement de fichiers de devis PDF générés par le logiciel 2020
  • Génération automatique de codes SKU OCCW standard basés sur des règles prédéfinies
  • Gestion des tables de prix OCCW standard avec support d'importation au format Excel
  • Support d'exportation vers Excel, CSV, PDF et autres formats
  • Support pour les langues d'interface chinoise, anglaise et française
  • Fonctionnalité de création manuelle de devis
  • Gestion du mappage SKU (administrateurs)
  • Requêtes de prix en temps réel
  • Gestion des paramètres système, y compris la bascule de connexion utilisateur, la modification du mot de passe administrateur, etc.
  • Fonctionnalité de modification du mot de passe administrateur avec support de mise à jour sécurisée du mot de passe

Création Manuelle de Devis
Description de la Fonction

Lorsque vous devez créer des devis manuellement ou modifier des devis existants, vous pouvez utiliser la fonction de création manuelle de devis. Cette fonction fournit un moyen flexible de construire des devis étape par étape.

Étapes d'Opération
1
Sélectionner Type Produit

Choisir parmi les catégories de produits disponibles

2
Ajouter Produits

Ajouter les produits un par un avec leurs spécifications

3
Définir Tarification

Configurer les informations de tarification et de quantité

4
Générer Devis

Générer et exporter le devis final

Analyse des Ventes
Fonctions d'Analyse

Le module d'analyse des ventes fournit des capacités d'analyse de données complètes pour vous aider à comprendre la performance des ventes et les tendances du marché.

Dimensions d'Analyse Disponibles
Analyse de Performance
  • Performance des ventes par période
  • Analyse du type de client
  • Comparaison de performance des vendeurs
  • Analyse des ventes par catégorie de produit
Analyse des Tendances
  • Tendances de ventes mensuelles/trimestrielles
  • Analyse du taux de conversion des clients
  • Distribution du montant des commandes
  • Comparaison des ventes régionales
Options d'Exportation de Données
  • Format Excel : Données détaillées avec graphiques et analyse
  • Format CSV : Données brutes pour traitement ultérieur
  • Format PDF : Rapports professionnels pour présentation
Questions Fréquemment Posées & Solutions
Problèmes Courants et Solutions

Veuillez vérifier les points suivants :

  • Le format de fichier est-il PDF ?
  • La taille du fichier dépasse-t-elle 16MB ?
  • Le fichier est-il corrompu ou chiffré ?
  • La connexion réseau est-elle normale ?

Vous pouvez essayer les méthodes suivantes :

  • Vérifier si le format de fichier PDF est standard
  • Ajuster les règles de mappage SKU
  • Éditer manuellement les résultats d'analyse
  • Contacter le support technique

Les paramètres système nécessitent des privilèges administrateur :

  • Se connecter avec un compte administrateur
  • Entrer dans la page des paramètres système
  • Modifier les paramètres de configuration pertinents
  • Sauvegarder les paramètres et redémarrer le service
