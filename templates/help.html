导入过滤设置（管理员功能）

管理员可以在"导入数据过滤"页面配置过滤条件：

  • 最小导入金额阈值：只导入总计金额大于等于设定值的数据
  • 销售人员过滤：多个销售人员用逗号分隔，留空表示不过滤
  • 客户过滤：多个客户用逗号分隔，留空表示不过滤
  • 日期范围：设置开始和结束日期，只导入指定时间范围的数据
  • 订单状态过滤：动态多选，系统自动提取数据中的所有订单状态
数据导入步骤
  1. 准备标准格式的 Excel 文件（销售订单.xlsx）
  2. 在主页面点击"选择销售订单文件"
  3. 上传文件，系统会自动应用预设的过滤条件
  4. 查看"导入数据过滤后剩余的行数"确认过滤结果
  5. 点击"更新分析表格"开始分析

查看导入数据

点击"查看导入数据"按钮可以：

  • 查看原始导入的销售数据
  • 支持列排序（点击列标题）
  • 支持搜索和过滤
  • 支持分页显示
  • 支持导出为Excel文件
查看转换数据

点击"查看转换数据"按钮可以：

  • 查看系统转换后的标准化数据
  • 包含新增字段：订单金额、报价单金额、订单数量、报价单数量
  • 支持所有查看导入数据的功能

销售员业绩分析表

显示每个销售员的业绩统计：

  • 订单数量：成功转化的订单数
  • 总报价数量：总报价单数量
  • 订单金额：订单总金额
  • 总报价金额：报价单总金额
  • 数量转化率：订单数量/总报价数量
  • 金额转化率：订单金额/总报价金额
客户订单分析表

显示每个客户的订单统计和转化率，默认显示10条记录。

订单金额转化率分析

双Y轴折线图：

  • 主Y轴（实线）：订单金额、总报价金额
  • 副Y轴（虚线）：金额转化率
  • 支持按销售人员过滤
订单数量转化率分析

双Y轴折线图：

  • 主Y轴（实线）：订单数、总报价数量
  • 副Y轴（虚线）：数量转化率
  • 支持按销售人员过滤
销售人员对比图表

包含三个对比图表：

  • 数量对比：对比不同销售人员的订单数量和报价数量
  • 金额对比：对比不同销售人员的订单金额和报价金额
  • 转化率对比：对比数量转化率和金额转化率
  • 支持图表最大化显示
  • 销售员业绩表格增加汇总行，显示全员统计
多语言支持

系统完整支持三种语言：

  • 中文：简体中文界面，适合国内用户
  • 英文：标准英语界面，面向国际用户
  • 法文：法语界面支持，适合加拿大法语区
  • 实时语言切换，无需重新登录
  • 所有界面元素完整翻译

转换规则

系统根据订单状态自动转换数据：

销售订单：
  • 订单金额 = 总计金额
  • 报价单金额 = 总计金额
  • 订单数量 = 1
  • 报价单数量 = 1
报价单：
  • 订单金额 = 0
  • 报价单金额 = 总计金额
  • 订单数量 = 0
  • 报价单数量 = 1
已发送报价单：
  • 订单金额 = 0
  • 报价单金额 = 总计金额
  • 订单数量 = 0
  • 报价单数量 = 1
已取消：
  • 自动过滤掉此类数据，不参与统计
转化率计算
  • 数量转化率 = 订单数量 ÷ 报价单数量 × 100%
  • 金额转化率 = 订单金额 ÷ 报价单金额 × 100%
金额区间过滤

系统支持按报价单金额进行区间过滤分析：

  • ≤1000：报价单金额小于等于1000的数据
  • 1000-5000：报价单金额在1000-5000之间的数据
  • 5000-10000：报价单金额在5000-10000之间的数据
  • ≥10000：报价单金额大于等于10000的数据
  • 全部金额：不限制金额范围
