转换规则

系统根据订单状态自动转换数据：

销售订单：
  • {{ t("help_order_amount") }} = 总计金额
  • 报价单金额 = 总计金额
  • {{ t("help_order_quantity") }} = 1
  • 报价单数量 = 1
报价单：
  • {{ t("help_order_amount") }} = 0
  • 报价单金额 = 总计金额
  • {{ t("help_order_quantity") }} = 0
  • 报价单数量 = 1
已发送报价单：
  • {{ t("help_order_amount") }} = 0
  • 报价单金额 = 总计金额
  • {{ t("help_order_quantity") }} = 0
  • 报价单数量 = 1
已取消：
  • 自动过滤掉此类数据，不参与统计
转化率计算
  • {{ t("help_quantity_conversion_y_axis") }} = {{ t("help_order_quantity") }} ÷ 报价单数量 × 100%
  • {{ t("help_amount_conversion_rate") }} = {{ t("help_order_amount") }} ÷ 报价单金额 × 100%
金额区间过滤

系统支持按报价单金额进行区间过滤分析：

  • ≤1000：报价单金额小于等于1000的数据
  • 1000-5000：报价单金额在1000-5000之间的数据
  • 5000-10000：报价单金额在5000-10000之间的数据
  • ≥10000：报价单金额大于等于10000的数据
  • 全部金额：不限制金额范围
