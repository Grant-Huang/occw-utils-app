System Overview
System Description

The OCCW Quotation System is a professional PDF quotation processing tool designed for Oppein Cabinet Canada West Ltd., used to convert quotations generated by 2020 software into standardized OCCW format.

Permission Description

Regular User Permissions: Can upload and parse PDFs, view results, export files, and view help documentation

Administrator Permissions: In addition to regular user permissions, can also manage OCCW price tables, configure SKU rules, and manage SKU mapping relationships

Main Features
  • Support for uploading PDF quotation files generated by 2020 software
  • Automatically generate OCCW standard SKU codes based on preset rules
  • Manage OCCW standard price tables with Excel format import support
  • Support export to Excel, CSV, PDF and other formats
  • Support for Chinese, English, and French interface languages
  • Manual quotation creation functionality
  • SKU mapping management (administrators)
  • Real-time price queries
  • System settings management, including user login toggle, administrator password modification, etc.
  • Administrator password modification functionality with secure password update support

Frequently Asked Questions & Solutions
Common Issues and Solutions

Please check the following points:

  • Is the file format PDF?
  • Is the file size over 16MB?
  • Is the file corrupted or encrypted?
  • Is the network connection normal?

You can try the following methods:

  • Check if the PDF file format is standard
  • Adjust SKU mapping rules
  • Manually edit parsing results
  • Contact technical support

System settings require administrator privileges:

  • Login with administrator account
  • Enter system settings page
  • Modify relevant configuration parameters
  • Save settings and restart service
